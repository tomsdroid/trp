Kebijakan Privasi TheraLog

Pembaruan Terakhir: 15 Juli 2026

Kami di TheraLog menghargai privasi Anda. Kebijakan ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi data pribadi serta data operasional Anda selama menggunakan aplikasi ini.

1. Data yang Kami Kumpulkan

Kami mengumpulkan data yang diperlukan untuk operasional pencatatan, termasuk namun tidak terbatas pada: ID Terapis, nama lengkap, penempatan cabang, dan rincian log sesi terapi (waktu, durasi, jenis layanan).

2. Penyimpanan Data Lokal

Untuk mendukung fitur luring (offline), TheraLog menyimpan data log sesi sementara di dalam memori peramban (IndexedDB/Local Storage) perangkat Anda. Anda bertanggung jawab untuk menjaga keamanan perangkat Anda dengan kata sandi atau kunci layar.

3. Sinkronisasi Data

Data yang tersimpan di perangkat akan dikirimkan secara otomatis atau manual ke peladen (server) pusat perusahaan melalui koneksi terenkripsi. Data ini akan digunakan semata-mata untuk keperluan evaluasi kinerja, rekapitulasi, dan pelaporan operasional manajemen.

4. Keamanan Data

Kami membatasi akses ke aplikasi menggunakan sistem PIN. Pastikan Anda tidak membagikan PIN Anda kepada pihak lain. Kami tidak akan menjual atau menyebarkan data log Anda kepada pihak ketiga di luar manajemen perusahaan.
