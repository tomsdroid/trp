Kebijakan Privasi

Terakhir diperbarui: 8 April 2026

PT Aura Gears ID (Perorangan) ("kami") berkomitmen untuk melindungi privasi dan data pribadi Anda sebagai pengguna aplikasi TheraLog. Dengan menggunakan aplikasi ini, Anda setuju dengan praktik pengumpulan dan penggunaan data yang dijelaskan dalam kebijakan ini.

1. Data Yang Kami Kumpulkan

Kami hanya mengumpulkan data yang diperlukan untuk operasional aplikasi:

  • Data Akun: ID Therapist, Nama, Cabang, Area, dan PIN yang dienkripsi
  • Data Log Terapi: Data sesi terapi yang Anda input untuk keperluan rekap dan statistik
  • Data Perangkat: Informasi dasar perangkat untuk keperluan debugging dan keamanan
2. Penggunaan Data

Data yang kami kumpulkan digunakan untuk:

  • Menyediakan dan memelihara layanan TheraLog
  • Menghasilkan rekap dan statistik terapi per cabang
  • Melakukan sinkronisasi data ke server kami
  • Meningkatkan keamanan dan kualitas aplikasi
3. Penyimpanan Data

Data Anda disimpan secara lokal di perangkat Anda menggunakan IndexedDB dan disinkronkan secara berkala ke server kami. Data di server dilindungi dengan enkripsi dan hanya dapat diakses oleh admin yang berwenang.

4. Keamanan Data

Kami menerapkan langkah-langkah keamanan untuk melindungi data Anda:

  • PIN 6 digit untuk mengakses aplikasi
  • Enkripsi data saat transmisi dan penyimpanan
  • Tidak menyimpan password dalam bentuk plain text
5. Berbagi Data

Kami TIDAK akan menjual, menyewakan, atau membagikan data pribadi Anda kepada pihak ketiga. Data hanya dibagikan ke manajemen Amarta & Rumah Spa untuk keperluan rekap internal.

6. Hak Anda

Anda berhak untuk:

  • Meminta akses ke data pribadi Anda
  • Meminta perbaikan data yang tidak akurat
  • Meminta penghapusan akun dan data Anda

Untuk permintaan di atas, silakan hubungi admin cabang atau email kami di bawah.

7. Perubahan Kebijakan

Kami dapat memperbarui kebijakan ini sewaktu-waktu. Perubahan akan diinformasikan melalui aplikasi. Versi terbaru selalu tersedia di halaman ini.
